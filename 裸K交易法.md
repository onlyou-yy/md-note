## 突破信号

什么是突破？当K线突破上一个高点的时候就是突破

![PixPin_2025-11-21_20-15-03](./裸K交易法/PixPin_2025-11-21_20-15-03.png)

### 确定初步突破(Initial Break,IB)

在上升趋势里，当最高的蜡烛被反向突破，那这个区域就是初步突破区域。

+ IB只是提供初步价格转向的可能性
+ IB并非入场信号，除非是以Dominant Break(**DB**)的形态出现。

![PixPin_2025-11-21_20-18-54](./裸K交易法/PixPin_2025-11-21_20-18-54.png)

### 蜡烛突破口（CandleBreak，CB)

再出现 **IB** 之后还需要等待K线突破 **CB** 才可以，这个信号才是有效的

![PixPin_2025-11-21_20-23-27](./裸K交易法/PixPin_2025-11-21_20-23-27.png)

**CB不可和IB处在同一个位置**

![PixPin_2025-11-27_13-34-04](./裸K交易法/PixPin_2025-11-27_13-34-04.png)

### Dominant Break ( DB)

由**3个或以上**的蜡烛形成的IB突破可以成为入场的信号

![PixPin_2025-11-21_20-31-09](./裸K交易法/PixPin_2025-11-21_20-31-09.png)

**所有IB，DB，和窗口突破，都是以蜡烛收价为标准**

### 如何察觉信号被市场否决？

当IB区域被反向突破，信号就不再成立。

![PixPin_2025-11-21_20-34-40](./裸K交易法/PixPin_2025-11-21_20-34-40.png)

## 使用斐波那契寻找止盈点

黄金比例（GoldenRatio），是以斐波纳契（Fibonacci）数列计算出来的完美比例

![PixPin_2025-11-21_20-43-40](./裸K交易法/PixPin_2025-11-21_20-43-40.png)

### 斐波纳契回撤的设定

![PixPin_2025-11-21_20-44-17](./裸K交易法/PixPin_2025-11-21_20-44-17.png)

**具体步骤**

1. 寻找蜡烛突破的信号，信号包括了CB的突破，或DominantBreak（DB)。
2. 使用斐波纳契回撤工具，把1.0定位在突破CB的蜡烛收价位置。
3. 把0定位在整个信号最低价的位置。
4. 从黄金分割线呈现出的TP1（1.618),TP2(2.618),TP3（4.236）就是止盈点。

使用斐波纳契回撤拉出来的线如下，其中需要关注的是`TP1`、`TP2`、`TP3`三个是止盈点，`minor Entry`，`major Entry`是回撤点

![PixPin_2025-11-21_20-46-17](./裸K交易法/PixPin_2025-11-21_20-46-17.png)

![PixPin_2025-11-21_20-49-18](./裸K交易法/PixPin_2025-11-21_20-49-18.png)

对于**DB**来说，可以直接再DB的收价位开始拉

![PixPin_2025-11-21_20-50-29](./裸K交易法/PixPin_2025-11-21_20-50-29.png)

### 注意事项

+ TP3如果破了后，我们可以把Fibo1再调整至最高/最低价，然后寻找延伸的止盈点。
+ 如果TP3再破，就必须转换更大的周期图来寻找止盈的区域，比如H1转换H4，H4转换Daily
+ 入场做反向的信号如果是发生在TP2/TP3区域，将有比较大的概率。

![PixPin_2025-11-27_18-13-41](./裸K交易法/PixPin_2025-11-27_18-13-41.png)



## 寻找最佳入场点

信号成立后，等价格回撤在进场开单，以取得更好的RR。通过4种策略，寻找最理想的位置挂单：

1. 从蜡烛形态寻找A-B-C波段，挂单在B位置
2. 挂单在IB区域
3. 转去小周期图寻找回撤终点
4. 利用斐波纳契(Fibonacci)回撤工具挂单

***注意：一旦IB区域被反向突破，必须止损。***

#### 从蜡烛形态寻找A-B-C波段

什么是A-B-C波段？

![PixPin_2025-11-27_18-20-48](./裸K交易法/PixPin_2025-11-27_18-20-48.png)

在B的位置挂单，等价格回撤入场。注意在找A-B-C的时候**C的位置一定要比A的位置低或高**，C点一般都是突破了DB的，其实认真观察会发现每个B点都是IB

#### 挂单在IB区域

IB是信号的最后关口，最简单的入场方法，可以拿到最好的RR。但是价格不一定会回来这个区域，也需克服恐惧心理

![PixPin_2025-11-28_19-42-18](./裸K交易法/PixPin_2025-11-28_19-42-18.png)

#### 转去小周期图寻找回撤终点

操作上有些难度比较适合有经验者，当有信号时，转去小周期图寻找回撤目标

+ 在H4有信号，转去H1寻找回撤
+ 在H1有信号，转去M30/M15寻找回撤

![PixPin_2025-11-28_19-45-45](./裸K交易法/PixPin_2025-11-28_19-45-45.png)

**高风险做法**：直接在TP1/2/3区域挂单；**低风险做法**：等待出现做多信号才开单

#### 利用斐波纳契(Fibonacci)回撤工具挂单

具体步骤：

1. 等待信号出现
2. 等价格回撤
3. 把黄金分割线移至波段的最高/低
4. 在Minor/Major区域挂单

### 动量震荡指标（AO），简称AO

动量震荡指标（Awesome Oscillator），简称**AO**（就是**MACD**，只不过不看那两根线），计算方法：`AO = SMA（中间价，5）- SMA（中间价，34）`,`中间价 =（最高价+最低价）/ 2`，**用来判定市场的动量，也就是推动力**，主要是寻找有**背离**的信号，对入场的风险做出调整。

#### 什么是背离(Divergence)?

+ 当价格创新低，但AO并没有相应创新低（牛市背离），视为利好（利多）讯息，价格跌势或将完结。
+ 相反，若价格创新高，但AO并没有相应创新高（熊市背离），视为利淡（利空）讯息，价格涨势或将完结。
+ 有背离的反向信号，价格来到TP2/TP3的概率也比较高。
+ 没有背离的反向信号，通常被视为暂时性的价格调整，信号被市场否决的概率也比较高。
